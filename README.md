# stock-tracker

Simple app that allows fetching and displaying stock market data for companies of choice

[Link to live demo](https://natalia-stock-tracker.surge.sh/)


## Tools

* React 16
* Webpack 4
* Babel 7
* Material UI 4

## How to install and develop?

* npm install
* npm start
* visit `http://localhost:8080/`

## How to build?  

* npm run build
* check dist folder
